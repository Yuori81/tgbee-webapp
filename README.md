# TG Bee — Telegram Mini App

Сервис аналитики Telegram-каналов.
20 экранов, дизайн-система Unbounded + Geologica, тёмная тема.

## Как запустить (пошагово для Windows)

### 1. Открой папку в Cursor

Открой Cursor → File → Open Folder → выбери папку `tgbee-cursor`

### 2. Установи зависимости

Открой терминал в Cursor (Ctrl + `) и напиши:

```
npm install
```

Подожди пока скачается (1-2 минуты). Появится папка `node_modules`.

### 3. Запусти проект

```
npm run dev
```

В терминале появится ссылка типа `http://localhost:3000` — открой её в браузере.

### 4. Готово!

Ты увидишь приложение — такое же как в Claude, но теперь оно работает у тебя локально.

## Структура проекта

```
tgbee-cursor/
├── index.html          ← Точка входа (HTML + Telegram SDK)
├── package.json        ← Зависимости проекта
├── vite.config.js      ← Настройки сборщика
├── src/
│   ├── main.jsx        ← Запуск React
│   ├── App.jsx         ← ВСЁ приложение (20 экранов + навигация)
│   ├── components/
│   │   └── UI.jsx      ← Общие компоненты (кнопки, хедер, табы)
│   └── styles/
│       ├── global.css  ← Глобальные стили и анимации
│       └── tokens.js   ← Цвета и шрифты
```

## Что внутри (20 экранов)

### P1 — Ядро
- Онбординг (P1-3)
- Первый анализ с прогрессом (P1-4)
- Результат анализа с метриками (P1-5)
- Кабинет с карточками (P1-6)
- Разбор канала и воронки (P1-7)
- Поиск каналов (P1-8)
- Профиль + Stars + рефералка (P1-9)

### P2 — ВП и Реклама
- Список ВП-партнёров с фильтрами (P2-1)
- Карточка ВП-партнёра (P2-2)
- Список рекламных площадок (P2-3)
- Карточка рекламной площадки (P2-4)

### P3 — Аналитика
- Отчёт по росту с источниками (P3-1)
- Форма отзыва ВП/Реклама (P3-2)
- Бейджи и кейсы (P3-3)

### PA — Воронка агентства
- Анкета 6 шагов (PA-2)
- Заявка принята (PA-3)
- Статус заявки (PA-4)

### Дополнительно
- Stars: магазин + оплата + успех + ошибка
- Настройки уведомлений
